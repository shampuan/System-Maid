IMPROVEMENTS:

1. The software icon/logo has been updated.

2. The interface has been colored to create a more aesthetically pleasing GUI.

3. The File System Cleaning category has been added:

* Delete broken links
* Clean empty directories
* Delete Snap/Flatpak user cache remnants

4. In the System Optimization tab:

* Clean Swap Areas feature has been added.
* Swappines setting has been added.
* Corrected incorrect wording and description in power saving mode. (No longer 75%).

5. Translation issues/deficiencies have been fixed.

6. Important filled boxes are empty by default; the user determines what to select and clear.

Don't be fooled by the foreign language images. Also, don't be fooled by the dark theme; the system will adapt to your theme.
